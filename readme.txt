== Description ==
KasLatex is a latex language convert to image.

This plugins is, in other words, Latex to wordpress.

This is part of KasPlugins for Wordpress.
 
== Installation ==

Wordpress basic instalation.

Option 1
 
1. Download.
2. Unzip. 
3. Upload the wp-kaslatex folder to the plugins directory.
4. Activate the plugin.
5. Go to new KasPlugins menu.
6. Select Kaslatex, and set personal settings.

Option 2
 
1. Go to addnew plugins on Plugins menu.
2. Find wp-kaslatex. 
3. Press install.
4. Later, Activate the plugin.
5. Go to new KasPlugins menu.
6. Select KasLatex, and set personal settings.

= Upgrade =

1. Press upgrade plugin in wordpress plugins list when plugins have a new version.
